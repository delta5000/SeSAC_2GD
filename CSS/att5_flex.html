  1. CSS3부터 나타난 방식
  2. 화면 레이아웃을 구성할 때 가장 많이 사용하는 방법
  3. 부모요소(컨테이너)에 flex를 선언하면 자식 요소의 레이아웃이 변경됨
  • flex 속성을 갖는 요소, flex 부모 요소
  • 컨테이너에 적용 가능한 속성이 있음
    1. flex-direction
      1. 기본설정은 row 수평정렬 (좌->우)
      2. row-reverse 수평정렬 (우->좌)
      3. columm 수직정렬 (위->아래)
      4. columm-reverse 수직정렬 (아래->위)
    2. flex-wrap
      • 줄바꿈 할지 말지 정하는 속성이다.
      1. no-wrap 기본설정. 줄바꿈 안함
      2. wrap 줄바꿈함
      3. wrap-reverse 줄바꿈을 하는데 아이템을 역순으로 배치
    3. justify-content
      1. 주축에 대한 정렬 방법. 아이템들을 어떻게 배치시킬 것 인지.
      2. flex-start : item들을 주 축의 시작지점 기준으로 나열
      3. flex-end : 아이템들을 주 축의 끝 지점 기준으로 나열
      4. center:주축 방향으로 중앙 정렬
      5. space-between : 처음과 마지막 아이템들을 양 끝에 붙이고 나머지영역 똑같이 나눠가짐
      6. space-around : 양 끝 영역이 1이라면 아이템과 아이템 사이의 영역은 2
      7. space-evenly: 모든 영역 공평하게 나눠 가짐
    4. align-items
      1. 교차축에 대해서 아이템을 배치하는 방법이다.
      2. stratch 기본속성 아이템이 높이가 없을 때 컨테이너 내부에 할당된 교차축의 크기만큼 늘어난다.
      3. flex-start
      4. flex-end
      5. center
    5. align-content
      1. 뭉쳐 다니는 것이다.
      2. 교차축에 대한 여러줄 정렬 방법
      3. 속성은 저스티파이콘텐트랑 동일하다. 차이점은 저스티파이는 주축, 얼라인콘텐츠는 교차축.
      4. 플랙스-랩이 두줄 이상일 때 의미가 있다. 여러줄이 뭉쳐다니는 속성이기 때문에.
      5. flex-start : item들을 교차축의 시작지점 기준으로 나열
      6. flex-end : 아이템들을 교차축의 끝 지점 기준으로 나열
      7. center:교차축 방향으로 중앙 정렬
      8. space-between : 처음과 마지막 아이템들을 양 끝에 붙이고 나머지영역 똑같이 나눠가짐
      9. space-around : 양 끝 영역이 1이라면 아이템과 아이템 사이의 영역은 2
      10. space-evenly: 모든 영역 공평하게 나눠 가짐
